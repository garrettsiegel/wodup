# WOD Creator

Generate CrossFit-style / functional fitness workouts based on equipment, time, intensity, skill level, and goals.

## Stack

- Vite + React 19 + TypeScript
- Tailwind CSS v4
- Zod

## Dev

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build |
| `npm run lint` | Run ESLint |
| `npm run typecheck` | Run TypeScript compiler check |
| `npm run verify` | Lint + typecheck |
